# Collab Connect Bot

MVP Telegram-бота, который соединяет бизнес и креаторов.

## Что умеет

- регистрация бизнеса
- регистрация креатора
- создание заявки бизнесом
- просмотр доступных заявок креатором
- отклик креатора на заявку
- принятие / отклонение отклика бизнесом
- открытие контактов после принятия
- простая админ-панель
- SQLite база `collab_bot.db`

## Установка

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
```

В `.env` вставьте токен бота и Telegram ID админа:

```env
BOT_TOKEN=123456:ABC
ADMIN_IDS=123456789,987654321
```

## Запуск

```bash
python bot.py
```

## Команды

- `/start` — главное меню
- `/admin` — админ-панель

## Логика

1. Бизнес создаёт заявку.
2. Креаторы видят активные заявки и нажимают «Откликнуться».
3. Бизнес получает карточку креатора.
4. Бизнес нажимает «Принять» или «Отклонить».
5. Если принял — бот открывает контакты обеим сторонам.
