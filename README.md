# КЛИК | Медиа-маркет — MVP

## Railway Variables
Добавьте:
- BOT_TOKEN = токен Telegram-бота
- ADMIN_IDS = ваш Telegram numeric ID (необязательно для обычного запуска; нужен для /admin)

## Railway Start Command
Если Railway не определит команду автоматически, в Settings -> Deploy -> Start Command укажите:
python bot.py

## Проверка
После успешного деплоя отправьте боту /start.
