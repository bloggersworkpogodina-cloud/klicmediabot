# КЛИК | Медиа-маркет — v9

Добавлено фото профиля для блогеров и контент-креаторов.

- фото запрашивается сразу после имени;
- сохраняется Telegram `file_id` в PostgreSQL;
- новые деплои фото не стирают;
- при повторном редактировании профиля фото можно заменить;
- фото показывается в карточке профиля, медиа-базе и при новом отклике бизнесу;
- старые профили без фото продолжают работать;
- остальная логика v8 не менялась.

Railway Variables без изменений:
BOT_TOKEN
ADMIN_ID
DATABASE_URL
